# 🤖 Sorbonne AI Multi-Hub : Unified Chat Interface

Ce projet est une plateforme de discussion universelle développée avec **Python** et **Streamlit**. Elle permet de basculer instantanément entre les meilleurs modèles du marché (**OpenAI**, **Google Gemini**, **Groq**) au sein d'une interface unique, fluide et réactive.

🔗 **Lien du ChatBot :** [Accéder à l'application](https://chatbotmultiplechoices-mh345h7vv83eedduvfpzbe.streamlit.app/)

---

## 🌟 Points Forts & Fonctionnalités

* **Multi-Fournisseur :** Support natif pour **GPT-4o**, **Gemini 1.5 Pro** et **Llama 3.3** via une interface de sélection dynamique.
* **Comparaison en Temps Réel :** Idéal pour tester la pertinence et la précision des réponses entre différents modèles sur une même problématique.

---

## 🛠️ Stack Technique & Architecture

1. **Frontend :** **Streamlit** (Composants Chat avancés et gestion d'état réactive).
2. **Orchestration :** **LangChain** (Utilisation des méthodes `invoke` unifiées pour tous les fournisseurs).
